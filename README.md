# Parkinson Telemonitoring
This is the repository of the final project for CS114, focused on Parkinson telemonitoring. The dataset can be found [here](https://archive.ics.uci.edu/dataset/189/parkinsons+telemonitoring). The complete pipeline is implemented in, obviously, `complete_pipeline.ipynb`. No extra download is required for running `complete_pipeline.ipynb`, you can just download the notebook and it's ready to go. Other files and folder include:

- `models`: Store the best models for each target after training, evaluating and tuning.

- `Old_model`: Old model building pipeline created by [my teammate](https://github.com/BuiThiThanhTrang). No longer in use.

- `preprocessed_data`: Preprocessed data in .csv format
